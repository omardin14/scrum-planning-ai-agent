Project Analysis

Datadog Bits AI SRE
Enable Datadog Bits AI SRE Agent to investigate P1, P2 and P3 Monitor notifications and provide support via Slack. Engineers can interact with the agent via Slack, the agent has access to the Azure DevOps code and Confluence RunBooks are reviewed and for added context.
Greenfield2-week sprints &times; 2 sprints = ~4 weeks total
